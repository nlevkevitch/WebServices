Read me - ICTGWS - School of ICT Graded Work Schedule system

This app is a foundation for the Fall 2012 Web Services Architecture programming assignment.

Use at as a base for your work.

It will work as-is. During startup, it will create a database in your local SQL Server Express instance, called ICTGWS.

It will then add starter data for Programs, Subjects, and Employees.

You can see the results by requesting the "api/about" resource (or by using SQL Server Management Studio).
